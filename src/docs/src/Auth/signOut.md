---
title: puter.auth.signOut()
description: Sign out the current user from your application.
platforms: [websites, apps]
---

Signs the user out of the application.

## Syntax

```js
puter.auth.signOut()
```

## Parameters

None

## Return value

None

## Example

```html;auth-sign-out
<html>
<body>
    <script src="https://js.puter.com/v2/"></script>
    <script>
        puter.auth.signOut();
    </script>
</body>
</html>
```
