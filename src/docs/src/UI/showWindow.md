---
title: puter.ui.showWindow()
description: Shows the window of your application.
platforms: [apps]
---

The `showWindow` method allows you to show the window of your application.

## Syntax

```javascript
puter.ui.showWindow()
```

## Parameters

None.

## Return Value

None.

## Example

```html
<html>
<body>
    <script src="https://js.puter.com/v2/"></script>
    <script>
        puter.ui.showWindow();
    </script>
</body>
</html>
```
