---
title: puter.ui.hideWindow()
description: Hides the window of your application.
platforms: [apps]
---

The `hideWindow` method allows you to hide the window of your application.

## Syntax

```javascript
puter.ui.hideWindow()
```

## Parameters

None.

## Return Value

None.

## Example

```html
<html>
<body>
    <script src="https://js.puter.com/v2/"></script>
    <script>
        puter.ui.hideWindow();
    </script>
</body>
</html>
```
